# Count Number Of Subarry With Equal No Zero And One

&nbsp;

#### Given an array arr[] of size n containing 0 and 1 only. The problem is to count the subarrays having an equal number of 0’s and 1’s.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
arr = [1, 0, 0, 1, 0, 1, 1]
```

&nbsp;

**Output**

```
8
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
arr = [1, 0, 0, 1, 1, 0, 0, 1]
```

&nbsp;

**Output**

```
12
```

&nbsp;
**Example 3:**

&nbsp;

**Input**

```
arr = [1,1,1,1,0]
```

&nbsp;

**Output**

```
1
```
